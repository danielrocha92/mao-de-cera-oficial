# Mão de Cera Oficial

Este é o projeto de e-commerce para a Mão de Cera Oficial, construído com Next.js, Firebase e outras tecnologias modernas.

## Começando

Primeiro, instale as dependências:

```bash
npm install
```

Depois, rode o servidor de desenvolvimento:

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
